Règles du Phrasor

Glisse les briques vertes depuis la palette vers la ligne centrale pour construire ta formule. Fais-les glisser pour les réordonner ou clique sur × pour supprimer.

  • D : Déterminant
  • N : Nom
  • V : Verbe
  • Adj : Adjectif
  • Adv : Adverbe
  • Prep : Préposition
  • C : Connecteur
  • Rel : Relatif / pronom
  • X : Néologisme
  • Littéral : ajoute ton propre texte via le champ gris.

Chaque symbole génère un mot aléatoire. Fais défiler, cliquer/glisser ou toucher/glisser un mot pour changer. Double-clique pour muter au hasard. La ponctuation se colle automatiquement au mot précédent.
